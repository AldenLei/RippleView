# RippleView

Android 自定义View一个控件搞定多种水波纹涟漪扩散效果

> 博客地址：http://blog.csdn.net/zhuwentao2150/article/details/79452735

- 圆圈从中心可循环向外扩散
- 圆圈之间的扩散间距可以改变
- 可控制扩散圆的渐变度
- 圆圈可以是线条样式或者实心样式
- 圆圈扩散的速度可以控制
- 适配圆圈不同大小下的扩散效果
